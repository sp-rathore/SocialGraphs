This is the flag of the Democratic Republic of Afghanistan (1992-1996). This image has been converted to PNG from a en:TIFF that was included in the 1996 CIA World Factbook. A mirror of the TIFF can be found at http://dlhoffman.com/publiclibrary/factbook96/factbook/flag-tif/af.tif.

This flag image should be recreated using vector graphics as an SVG file. This has several advantages; see Commons:Images for cleanup for more information.
