A blank locator map for Great Britain. The idea is to superimpose Image:dot4gb.svg at the appropriate point. See w:Template:GBthumb for code to do this.

These two images can replace any locator map posted by Lupin - see w:Special:Contributions/LupinBot.

This is a vectorized version of Image:Gb4dot.png, with the scale changed from 200 km to 150 km/150 miles. The shoreline data come from the file gshhs_h.b from the GSHHS, and the rivers and the Irish border come from the CIA World DataBank II; both sources are in the public domain. Mercator projection.
