Summary

This image was created by myself ( Gilles Tran) with en:POV-Ray 3.6 using en:radiosity. The glasses, ashtray and pitcher were modeled with Rhino and the dice with en:Cinema 4D.

The POV-Ray code for this image is available here (zip file, 6 Mb). The code is also released into the Public Domain.
