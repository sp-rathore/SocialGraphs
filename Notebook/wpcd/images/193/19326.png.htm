Summary

Editing this file

This file is in SVG format. That means you can edit it with a text editor or an SVG-compatible graphics editor. However, you will find it better to use a text editor because otherwise your edits will be too imprecise.

Adding new data

If the data is between 1996 and 2006, just find that point in the file and update it. For example, at the moment the code for the usage statistics between 2000 and 2001 looks like this:

L 2000,83.75
L 2001,90

To insert data for the midpoint of 2000, just make it look like this:

L 2000,83.75
L 2000.5,86
L 2001,90

If you are adding data for before 1996 or after 2006, you'll need to change the transformations on the groups. This requires moderately detailed understanding of SVG. Seek help if you need it.
